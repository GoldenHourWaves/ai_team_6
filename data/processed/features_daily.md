# features_daily.csv 설명 문서

## 1. 파일 개요

`features_daily.csv`는 GDELT GKG 기반 비트코인 뉴스 데이터를  
**기사 단위가 아닌 ‘일(day) 단위’로 집계한 분석용 피처(feature) 테이블**입니다.

본 파일의 목적은 다음과 같다.

- 하루 동안 쏟아진 비트코인 관련 뉴스의 **양(Volume)**,
- 뉴스의 **전반적인 분위기(Tone)**,
- 뉴스가 다루는 **주제 구성(Themes)**

을 숫자로 요약하여  
**비트코인 일봉 가격 데이터와 직접 병합·분석**할 수 있도록 만드는 것입니다.

---

## 2. 행(Row)의 의미

- **1행 = 1일**
- 각 행은 해당 날짜 하루 동안의 모든 비트코인 관련 뉴스 기사를 집계한 결과입니다.
- 기사 개별 내용이나 제목이 아니라,  **“그날 뉴스 흐름 전체의 특성”**을 요약한다.

즉, 이 데이터는  
- 기사 1건을 설명하는 데이터가 아니라  
- **시장에 노출된 정보 환경(Information Flow)**을 설명하는 데이터입니다.

---

## 3. 컬럼(Column) 구성 및 의미

### 3.1 날짜 식별자

| 컬럼명 | 설명 |
|------|------|
| `date` | 집계 기준 날짜 (YYYYMMDD 형식). 비트코인 가격 데이터와 병합하기 위한 기본 키 |

---

### 3.2 뉴스 양(Volume) 관련 지표

| 컬럼명 | 설명 |
|------|------|
| `n_articles` | 해당 날짜에 수집된 비트코인 관련 기사 수 |
| `n_sources` | 해당 날짜에 등장한 고유 언론사(source) 수 |

**해석 포인트**
- `n_articles` 급증 → 특정 이슈 발생 가능성
- `n_sources`가 낮을수록 → 특정 이슈에 보도가 집중됨(쏠림 현상)

---

### 3.3 뉴스 분위기(Tone) 관련 지표

Tone 값은 GDELT GKG의 AvgTone을 기반으로 집계되었습니다.

| 컬럼명 | 설명 |
|------|------|
| `tone_mean` | 해당 날짜 뉴스 톤의 평균값 |
| `tone_std` | 톤 값의 표준편차 (뉴스 해석의 분산 정도) |
| `tone_neg_share` | 톤 값이 0보다 작은(부정적인) 기사 비율 |
| `tone_pos_share` | 톤 값이 0보다 큰(긍정적인) 기사 비율 |

**해석 포인트**
- `tone_mean < 0` → 전반적으로 부정적인 뉴스 환경
- `tone_neg_share` 상승 → 공포·리스크 뉴스 집중
- `tone_std` 상승 → 시장 해석이 엇갈림 (변동성 확대 가능성)

---

### 3.4 뉴스 주제(Theme) 빈도 지표

형식: theme_cnt__THEMENAME

예:
- `theme_cnt__REGULATION`
- `theme_cnt__MARKET`
- `theme_cnt__BLOCKCHAIN`

**의미**
- 해당 날짜에 기사들에서 해당 테마가 **몇 번 등장했는지**를 나타냄
- 기사 1건에 여러 테마가 포함될 수 있으므로,
  이는 기사 수가 아닌 **테마 등장 빈도**입니다.

**해석 포인트**
- 특정 테마 빈도 급증 → 해당 이슈가 시장의 주요 관심사였음을 의미
- 예:
  - `theme_cnt__REGULATION` 증가 → 규제 이슈 집중
  - `theme_cnt__MARKET` 증가 → 가격·시장 동향 중심 보도 증가

---

## 4. 데이터 생성 방식 요약

1. 기사 단위 데이터(`bitcoin_news_merged`)를 기준으로
2. 날짜(`date`)별로 groupby 수행
3. 기사 수, 톤 통계, 테마 빈도를 일별로 집계
4. 비트코인 가격 데이터와 동일한 일 단위 해상도로 정렬

---

## 5. 활용 목적 및 분석 예시

`features_daily.csv`는 다음과 같은 분석에 사용됩니다.

- 비트코인 일봉 가격과의 병합 분석
- 뉴스 톤 변화와 가격 변동의 상관관계 분석
- 뉴스 테마 구조 변화에 따른 가격 반응 비교
- 특정 이벤트 기간(예: 급락 구간)의 뉴스 흐름 설명

---

## 6. 핵심 요약

- `features_daily.csv`는 **뉴스 기사 요약 데이터가 아니라**
- **하루 동안의 뉴스 환경을 정량화한 분석용 피처 테이블**입니다.
- 각 행은 “하루의 정보 흐름”을 나타내며,
- 비트코인 가격 변동을 설명하기 위한 핵심 입력 데이터로 사용될 수 있습니다..
